Dropout層
目的：過学習防止
方法：ノードをランダムに無効化
適用時：隠れ層の出力時
推論時：全ノードを仕様（無効化）

バッチ正規化層
目的：勾配消失を防ぐ、学習の高速化、重みの初期値に対する依存を低減→学習の安定化、高速化
方法：各ミニバッチで正規化
適用時：角層の入力前または出力後
推論時：学習時の平均・分散を使用

各レイヤの出力を一定に保つことで学習をスムーズに！！！
